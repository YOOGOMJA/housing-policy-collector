# AI 가이드라인

- 구현 전후 관련 문서를 동기화합니다.
- 정책 위반 가능성은 즉시 경고합니다.
- 모호한 조건은 보수적으로 판단합니다.
- 코드 변경 시 린트 및 컨벤션 체크를 수행합니다.
- 문서 변경 시 YAML 파일에 한해 포맷/문법 체크를 수행합니다.
- 신규 문서 추가 시 문서 인덱스와 연관 문서 반영 누락을 확인합니다.
- 모든 문서는 한국어를 기본 작성 언어로 사용합니다(고유명사/기술용어 제외).
- PR 제목/본문/검증 결과는 한국어로 작성합니다.
- 충돌 발생 시 충돌 원인·해결 방식·재발 방지 조치를 PR 본문에 기록합니다.
